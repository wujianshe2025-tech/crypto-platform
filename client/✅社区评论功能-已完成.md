## 💾 数据存储

所有数据保存在浏览器的localStorage中：
- 键名：`community_posts`
- 格式：JSON数组
- 包含：帖子、点赞、评论

### 数据结构
```javascript
{
    id: 时间戳,
    user: '用户名',
    content: '帖子内容',
    likes: 点赞数,
    likedBy: ['user1', 'user2'],
    comments: [
        {
            id: 时间戳,
            user: '评论者',
            content: '评论内容',
            time: '时间'
        }
    ],
    time: '发布时间'
}
```

## 🔧 技术实现

### 使用的技术
- 纯JavaScript（无框架）
- localStorage API
- Tailwind CSS
- CSS动画

### 关键函数
- `renderPosts()` - 渲染帖子列表
- `createPost()` - 创建新帖子
- `toggleLike()` - 切换点赞
- `toggleComments()` - 展开/折叠评论
- `addComment()` - 添加评论
- `showToast()` - 显示提示
