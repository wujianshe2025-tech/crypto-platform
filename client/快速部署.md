# 🚀 快速部署到Vercel

## 最简单的部署方式

### 步骤1: 准备代码

确保你的代码已经推送到GitHub：

```bash
# 在crypto-platform目录下
git add .
git commit -m "更新客户端为真实数据版本"
git push
```

### 步骤2: 部署到Vercel

#### 选项A: 使用Vercel网站（推荐新手）

1. 访问 https://vercel.com
2. 使用GitHub账号登录
3. 点击 "Add New Project"
4. 选择你的仓库
5. 配置设置：
   ```
   Framework Preset: Vite
   Root Directory: client
   Build Command: npm run build
   Output Directory: dist
   ```
6. 点击 "Deploy"
7. 等待几分钟，完成！

#### 选项B: 使用命令行（推荐熟练用户）

```bash
# 1. 安装Vercel CLI
npm install -g vercel

# 2. 登录
vercel login

# 3. 进入client目录
cd crypto-platform/client

# 4. 部署
vercel

# 5. 生产部署
vercel --prod
```

### 步骤3: 访问你的网站

部署完成后，Vercel会给你一个网址，例如：
```
https://your-project.vercel.app
```

## 🎯 一键部署按钮

如果你想让别人也能一键部署，可以在README中添加：

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/你的用户名/你的仓库名&project-name=crypto-platform&root-directory=client)

## ⚡ 自动部署

配置后，每次推送到GitHub，Vercel会自动部署：

1. 在Vercel项目设置中
2. 找到 "Git" 选项
3. 确保 "Production Branch" 设置为 `main` 或 `master`
4. 每次推送代码，自动部署！

## 🔧 常见问题

### Q: 部署后页面空白？
A: 检查Vercel的构建日志，确认没有错误

### Q: 路由不工作？
A: 确保有正确的重写规则（Vite项目通常自动处理）

### Q: API调用失败？
A: 检查浏览器控制台，确认API可访问
