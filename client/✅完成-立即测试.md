### 第3步：部署到Vercel（5分钟）

1. 访问 https://vercel.com
2. 登录你的账号
3. 点击 "Add New Project" 或 "Import Project"
4. 选择你的GitHub仓库
5. 配置项目：
   ```
   Project Name: crypto-platform (或你喜欢的名字)
   Framework Preset: Vite
   Root Directory: client
   Build Command: npm run build
   Output Directory: dist
   ```
6. 点击 "Deploy"
7. 等待2-3分钟
8. 获得你的网站地址！
