## 📊 技术细节

### 构建过程：
```javascript
// 简单的文件复制，不需要编译
node -e "const fs=require('fs');
  if(!fs.existsSync('dist'))
    fs.mkdirSync('dist');
  fs.copyFileSync('index.html','dist/index.html')"
```

### 为什么这样做：
- 我们的网站是纯HTML+JavaScript
- 不需要React编译
- 不需要TypeScript转译
- 直接复制HTML文件到dist目录
- Vercel直接提供静态文件服务
