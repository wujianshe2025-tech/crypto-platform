### 🔧 技术实现

#### 使用的API

1. **CoinGecko API**
   ```
   https://api.coingecko.com/api/v3/coins/markets
   ```
   - 免费使用，无需API密钥
   - 限制：每分钟10-50次请求
   - 返回：加密货币价格和市场数据

2. **CryptoCompare API**
   ```
   https://min-api.cryptocompare.com/data/v2/news/
   ```
   - 免费使用，无需API密钥
   - 限制：每秒20次请求
   - 返回：加密货币新闻

#### 自动刷新机制

```javascript
// 每30秒自动刷新价格数据
setInterval(() => {
    if (当前在行情页面) {
        loadCryptoPrices();
    }
}, 30000);
```
