# 加密货币平台 - 纯前端演示版

## 🎯 功能特性

✅ **实时行情** - 模拟加密货币价格实时更新
✅ **新闻资讯** - 分类展示（实时/重要/突发），智能标注利多/利空
✅ **社区论坛** - 发帖、评论、点赞互动
✅ **预测投票** - 创建预测、参与投票、查看结果

## 🚀 快速开始

### 在线预览（推荐）

#### StackBlitz
1. 访问 https://stackblitz.com/
2. 点击 "Import Project"
3. 上传 `demo` 文件夹
4. 自动运行，立即预览

#### CodeSandbox
1. 访问 https://codesandbox.io/
2. 拖拽 `demo` 文件夹到页面
3. 等待安装依赖
4. 在线预览

### 本地运行

```bash
cd demo
npm install
npm run dev
```

访问 http://localhost:5173

## 📦 技术栈

- React 18
- TypeScript
- Tailwind CSS
- React Router
- Vite

## 💡 演示说明

这是纯前端演示版本，使用模拟数据：

- 行情数据每3秒自动更新
- 新闻数据为预设内容
- 社区帖子可以发布和点赞（本地存储）
- 预测投票可以创建和参与（本地存储）
